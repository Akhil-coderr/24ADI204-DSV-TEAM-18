# Olympic Data Insights: A Data Science, Visualization, and Geopolitical Storytelling Project
# DSV_TEAM-18

A data science project that explores and analyzes Olympic athlete data to uncover meaningful patterns, trends, and insights through analytical storytelling.
Using exploratory data analysis and compelling visualizations, we transform complex raw data into clear narratives about athlete performance, global sporting trends, and geopolitical dynamics that can help sponsors, sports investors, and analysts make informed decisions.
